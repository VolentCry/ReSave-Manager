<h1 align="center">ReSave Manage</h1>
This program is a simple save manager for your video gamesüéÆ  

With this application, you can configure backup settings for your game saves‚Äîhow often they will be created and how many will be stored.  

The app is primarily aimed at pirated gaming, as services like Steam or even Hydra Launcher have their own cloud storage. This solution simply creates backup copies of available game saves so that, if necessary, you can always roll back your game‚Äîfor example, if you get stuck in a softlock trap‚Äîensuring you don‚Äôt lose all your progress. This feature is especially relevant for games that don‚Äôt have multiple save slots, such as the well-known game *Elden Ring*.  

In the future, the plan is to create cloud storage for your game saves by integrating with popular cloud services like Yandex Disk or Google Drive.  

### **Usage Instructions**  
To use the program, simply download the .exe file or the entire program in .zip format, then place it in any folder. The program will automatically create a separate folder for all future backups.  

Upon first launch, the app will prompt you to specify system folders where your games are stored. If your games are scattered across the disk, you can simply specify the root folder of the drive, though this will increase the search time for game saves.  

However, it‚Äôs worth mentioning that you can always manually specify everything. The program includes necessary utilities for this, though it will still take some time :)  

Each game gets its own folder, containing subfolders with all your backup saves.  

To configure the frequency and number of backups, there is a settings button next to each game on the main screen‚Äîthis is where everything is managed.  

The core functionality can be summarized as follows:  
1. **Backup frequency**  
2. **Number of backups**  
3. **Storage limit**  
4. **"Smart" backup**  

- *Backup frequency* ‚Äî You can set a time interval from 1 hour to 1 year.  
- *Number of backups* ‚Äî Sets a limit on how many backup copies are stored.  
- *Storage limit* ‚Äî Sets a limit on how much disk space a single game‚Äôs backups can occupy.  
- *"Smart" backup* ‚Äî Ensures the program won‚Äôt create backups if the user hasn‚Äôt played the game longer than the set backup frequency.  

In addition to the features listed above, the game settings also allow you to:  
- Open the game‚Äôs directory  
- Open the backup save directory  
- Open the current save directory  

All the settings mentioned above can be applied uniformly to all games at once if needed. To do this, go to the app‚Äôs general settings, select *"Apply to all"* (enabled by default), and set the desired parameters.
